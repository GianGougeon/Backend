# desafio 18

## E-commerce backend

1. Instalación

    ```sh
    npm install
    ```

    ### Dependencias utilizadas:

    - bcrypt
    - compression
    - connect-mongo
    - cookie-parser
    - dotenv
    - express
    - express-handlebars
    - express-session
    - mongoose
    - multer
    - nodemailer
    - passport
    - passport-local
    - twilio
    - winston

    ### devDependencias utilizadas:

    - sass
    - dotenv

2. Usar

    ```sh
    npm start
    ```

    El proyecto, que estará en http://localhost:8080

3. Subido en Heroku

    [Heroku](https://terceraentregagian.herokuapp.com/)

# desafio 24

## E-commerce backend

1. Instalación:

    ```sh
    https://deno.land/
    ```

2. Usar

    ```sh
    deno run server.ts
    ```

    El proyecto, que estará en http://localhost:8080/api/productos

3. Peticiones

    - GET /api/productos

    - GET /api/productos/:id

    - POST /api/productos

    - PUT /api/productos/:id

    - DELETE /api/productos/:id
